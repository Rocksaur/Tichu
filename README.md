# Tichu card counter by 야옹열

티츄(Tichu) 플레이 중 **이미 나온 카드**를 빠르게 체크해서 남은 카드를 추적할 수 있는 초경량 웹앱입니다.

이 버전은 특히 **iPhone Safari / 카카오톡 내장 브라우저**처럼 제한이 있는 모바일 환경에서도 최대한 안정적으로 동작하도록 단순한 구조로 만들어졌습니다.

## 주요 기능

- 숫자 카드 `2 ~ A`를 **각 4장 기준**으로 카운트
- 탭할 때마다 `4 → 3 → 2 → 1 → 0`으로 감소
- `0`이 된 숫자 카드는 다시 눌러도 되돌아가지 않음
- 특수 카드 `1 / 개 / 봉 / 용` 체크 가능
- 특수 카드도 체크 후 다시 눌러도 취소되지 않음
- `↶ 취소` 버튼으로 직전 입력 되돌리기
- `전체 초기화` 버튼으로 즉시 초기화
- 현재 카운트 상태 자동 저장
- `후원하기` 버튼 및 계좌번호 복사 기능 포함

## 파일

GitHub에 올릴 대표 파일은 아래 HTML 파일입니다.

- `tichu_counter_kakao_safe_compact_height_title19_summarybig_h1margin_resetdirect_simpler_donate_account_copy_nosectiontitles_by_smaller_special_nountoggle_specialblue.html`

원한다면 파일명을 더 짧게 바꿔서 예를 들어 아래처럼 사용해도 됩니다.

- `index.html`

GitHub Pages로 배포할 예정이라면 **파일명을 `index.html`로 바꾸는 것**을 추천합니다.

## 사용 방법

### 로컬에서 실행

별도 설치 없이 HTML 파일을 브라우저에서 열면 됩니다.

### iPhone에서 사용

1. HTML 파일을 Safari로 엽니다.
2. 공유 메뉴를 누릅니다.
3. **홈 화면에 추가**를 선택하면 앱처럼 사용할 수 있습니다.

### GitHub Pages로 배포

1. 이 저장소에 HTML 파일을 업로드합니다.
2. 대표 HTML 파일명을 `index.html`로 변경합니다.
3. GitHub 저장소의 **Settings → Pages**로 이동합니다.
4. Branch를 선택해 Pages를 활성화합니다.
5. 생성된 주소로 접속하면 바로 사용할 수 있습니다.

## 화면 구성

- 상단 제목
- 전체 카운트 표시 (`전체 x/56`)
- 특수 카드 버튼
- 숫자 카드 버튼 (`2~A`)
- 기능 버튼 (`↶ 취소`, `전체 초기화`)
- 후원하기 버튼 / 계좌 안내

## 기술적 특징

이 앱은 다음을 우선해서 설계되었습니다.

- 외부 라이브러리 없음
- 단일 HTML 파일
- 단순한 CSS / JavaScript 구조
- 모바일 내장 브라우저 호환성 우선

그래서 일반적인 웹앱보다 구조가 단순하지만, 게임 중 빠르게 쓰기에는 오히려 더 적합합니다.

## 주의

- 브라우저 환경에 따라 자동 저장(localStorage) 동작이 일부 제한될 수 있습니다.
- 카카오톡 내장 브라우저에서는 기기/버전에 따라 일부 동작이 다를 수 있습니다.
- 가장 안정적인 사용 환경은 **Safari**입니다.
